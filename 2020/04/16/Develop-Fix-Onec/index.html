兩種顯影定影合一沖洗配方

兩種顯影定影合一沖洗配方

四月 16, 2020
  • 定影劑
  • 顯影劑

本文將介紹兩種顯影定影合一的黑白菲林沖洗劑，配方來自期刊《理化檢驗》1979年04期。
作者用這兩種沖洗液沖洗使用柯達菲林拍攝的金相顯微照片，博主查了一下資料，金相顯微一般使用全色黑白菲林，而且要求顆粒細膩。故推測這兩個配方可以用於沖洗普通攝影照片，而且應該可以獲得比較細膩的顆粒。

警告：對苯二酚有毒，請做好相關防護工作，不要直接接觸對苯二酚，更不要食用！

警告：氫氧化鈉有強腐蝕性，使用時務必做好防護工作！

注意：米吐尓接触易发生过敏，请做好防护！對水生生物有毒且會危害水質，顯影後廢液請不要直接排入自然水體！

注意：溴化鉀對消化道粘膜有刺激性和神經抑制作用，使用時請做好防護！

配方一

藥品 質量 備註
蒸餾水 800ml
米吐爾 10g
無水亞硫酸鈉 50g 約0.4mol，也可使用100g七水合亞硫酸鈉
對苯二酚 40g
十二水合硫酸鋁鉀 20g
氫氧化鈉 35g
硫代硫酸鈉 110g
6-硝基苯骈间二氮茂 20ml 無法找到這個藥品的資料，可能原文有誤，看名字應該是類似柯達1#抗灰霧劑的藥品，根據原文介紹可用稍多的溴化鉀替代

加水至1000ml

配方二

藥品 質量 備註
蒸餾水 400ml
菲尼酮 0.2g
對本二酚 4g
十二水合硫酸鋁鉀 4g
無水碳酸鈉 5g 約0.05mol，也可使用14g十水合碳酸鈉
硫代硫酸鈉 12g
溴化鉀 1g
硼酸 2g
硼砂 2g

加水至500ml

使用方法

  1. 在20°C下。顯影3分鐘左右即可。
  2. 配方二效力略強於配方一，且顯影效果更好。
  3. 顯影定影合一沖洗劑有減感作用，拍攝時注意補償。
  4. 必須使用防灰霧劑（溴化鉀）。
  5. 藥劑保存性較差，最好現配現用，注意保存環境。
